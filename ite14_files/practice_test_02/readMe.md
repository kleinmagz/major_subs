Doubly linked list practice

Using the given program as a basis (Feb-10-Practice.c),
develop a menu-driven program that utilizes functions to perform the following operations:

1. Add a node at the beginning.
2. Add a node in the middle.
3. Add a node at the end.
4. Delete the node at the beginning.
5. Delete a node from the middle.
6. Delete the node at the end.
7. Display all nodes from the first node to the last node.
8. Display all nodes from the last node to the first node.
